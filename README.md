# 🏖️ South Goa Resorts Guide

A single-page, mobile-friendly guide comparing **11 South Goa resorts** for a July
(monsoon-season) family trip — 8 adults and 4 two-year-olds across 4 families.

It scores each resort on food quality, beach access, monsoon (indoor) activities,
and baby/toddler support, then lays out three concrete booking strategies.

## Sections

- **Overview** — trip parameters and key findings from review analysis
- **Top Resorts** — the 4 best picks at a glance
- **All 11 Resorts** — full cards with pros, cons, and red flags
- **Comparison** — side-by-side scoring table
- **Recommendations** — three booking options + resorts to avoid

## Tech

Plain static HTML/CSS/JS — no build step, no dependencies. Just open `index.html`
in a browser, or deploy the folder to any static host.

## Local preview

```bash
# any static server works, e.g.
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

This site is deployed on Vercel as a static site (the repo root is served directly).
